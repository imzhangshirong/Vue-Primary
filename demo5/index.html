事件修饰符

  • .prevent => 使用preventDefault()
  • .stop => 使用stopPropagation()
  • .capture => 捕获模式
  • .self => 当自身触发（子元素不会触发）
  • .once => 触发一次（2.1.4新增）

以上的后缀都可以串联使用，很强大是不是，我们来看看更棒的

按键修饰符

  • .enter
  • .tab
  • .delete => 捕获 “删除” 和 “退格” 键
  • .esc
  • .space
  • .up
  • .down
  • .left
  • .right
  • .数字 => 对应的键值例如c, .67
  • .ctrl （2.1.0新增）
  • .alt （2.1.0新增）
  • .shift （2.1.0新增）
  • .meta => win键 （2.1.0新增）
